# SMAC Rust Project Roadmap - The Grand Strategy

This roadmap details the path to a full-featured SMAC engine, encompassing AI intelligence, UX refinement, deep mechanics, and system engineering.

## Batch 1: AI Intelligence & Diplomacy
1. **AI Personality Matrix**: Define distinct faction archetypes (Militarist, Tech, Diplomat).
2. **AI Negotiation Dialogue Engine**: Implement a state-machine for AI diplomacy.
3. **Advanced AI Defense**: Improve AI base-defense logic based on threat levels.
4. **AI Exploration Bias**: Add "Discovery" vs "Expansion" focus tags.
5. **AI Secret Project Competition**: Create competitive tension for wonder construction.
6. **Dynamic Diplomatic Relations**: Add attitude decay/growth based on player actions.
7. **Espionage/Probe Teams**: Implement a formal "Probe" action system.

## Batch 2: GUI/UX Refinement
8. **Research Tree Visualizer (Graph View)**: Interactive node-link graph for tech tree.
9. **Animated Event Log**: Popup notifications for global events.
10. **Rich Tile Tooltips**: Real-time tile yield breakdowns and modifiers.
11. **Unit Movement Path Preview**: Draw paths before movement.
12. **Base Production Queue Drag-and-Drop**: Interactive queue management.
13. **Global Logistics Overview**: Tracking trade links and maintenance.

## Batch 3: Core Gameplay Depth
14. **Base Facility Upkeep**: Maintenance logic for all 19 facilities.
15. **Unit Mineral Support/Upkeep**: Mineral drain for units beyond support limits.
16. **Social Engineering Conflicts**: Disable mutually exclusive policy combinations.
17. **Advanced Terraforming Depletion**: Thermal Borehole risk/depletion mechanics.
18. **Advanced Unit Abilities**: Drop Pods, Artillery Bombardment.
19. **Faction Specialization**: Unique starting tech paths and faction-specific bonuses.
20. **Secret Project Full-Effect Integration**: Mechanical boosts for every wonder.
21. **Base Health & Siege Mechanics**: Direct bombardment of facilities/pop.

## Batch 4: Infrastructure & Engineering
22. **Persistence/Save Migration**: Robust save-file version migration.
23. **Localization Infrastructure**: Externalize display strings.
24. **Content Validation CLI Expansion**: Automated static content analysis.
25. **AI Turn Profiling**: Performance tracking for AI decision loops.
26. **Scenario Editor**: Map and unit placement tools for testing.
27. **Save System Versioning**: Handle data structure evolution.
28. **Binary Asset Packing**: Migrate from JSON/data files to binary blobs.

## Batch 5: Experimental Mechanics
29. **Planet Toxicity Crisis Events**: Dynamic crises (Tidal Chaos, Dust Fall).
30. **Mind Worm AI (Native Life)**: Unique non-standard faction AI logic.
31. **Governance Override**: "Machine Polity" crisis on high AI dependence.
32. **Global Trade Network**: Visualization/Tracking of planetary energy flow.
33. **Final Transcendence Sequence**: Endgame project completion narrative.
34. **Research Overflow**: Carry-over points for new technology.
35. **Energy Credits System**: Global bank for energy and rush-building.
36. **Unit Prototype Upgrades**: Upgrading units in base production.
37. **Advanced Map Generators**: Custom presets (Arid, Fungus-heavy).
38. **Audio Engine Integration**: UI sound effects.
39. **Developer Console**: Cheat mode for testing.
40. **Modding API**: Scripting interface for runtime mods.
