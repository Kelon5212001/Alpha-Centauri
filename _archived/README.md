This directory contains preserved code, plans, tooling experiments, and snapshots that are intentionally out of the active workspace path.

Nothing under `_archived/` is part of the current `cargo check`, `cargo test`, runtime content validation, or GUI runtime path.

Archive layout:

- `bevy_prototype_src/`: retired Bevy-era prototype code
- `bundles/`: tarball-compressed preserved experiment buckets that no longer need expanded source-tree presence
- `project_spec/`: early project-specification text files and packaged spec artifacts
- `root_artifacts/`: obsolete files moved out of the repository root
- `source_snapshots/`: ad hoc source patch artifacts and retired scratch subcrates removed from active crate paths
- `sample_saves/`: preserved manual save examples removed from the active GUI save directory
- `workspace_backups/`: preserved workspace backup snapshots moved out of the repository root

Use the repository root `README.md` and `ROADMAP.md` for the live state of the active workspace.
